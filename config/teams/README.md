# SAT Secretary BOT Teams App Package

This folder contains the starter Teams app manifest template for the SAT Secretary BOT feasibility work.

## Files

- `manifest.template.json` - replace placeholders and package as `manifest.json`
- `color.png` - required by Teams app package, not included yet
- `outline.png` - required by Teams app package, not included yet

## Placeholder Values

Replace:

- `${TEAMS_APP_ID}`
- `${TEAMS_BOT_ID}`
- `${ENTRA_CLIENT_ID}`
- `${PUBLIC_BASE_URL}`
- `${PUBLIC_DOMAIN}`

Then zip `manifest.json`, `color.png`, and `outline.png` for upload to Teams Developer Portal or your tenant app catalog.
